TeamB 運動揪團網站

## 專案簡介
這是一個不同於一般社群媒體，提供專業運動揪團組織功能。 使用 React.js 開發前端介面，Node.js + Express 實作後端 API，並以 MySQL 作為資料儲存。使用者可以新增、查看與刪除留言，前後端分離設計，適合展示 CRUD 與 RESTful API 能力。

## 使用技術

### Frontend（前端）
- React.js
- Next.js
  
### Backend（後端）
- Node.js
- Express.js
- CORS

### Database（資料庫）
- MySQL
